Diferença entre .style e .classList:

Use .style para aplicar um estilo inline rápido (ex.: element.style.backgroundColor = 'red'), ideal para mudanças pontuais. Prefira .classList para alternar classes CSS reutilizáveis (ex.: element.classList.toggle('estilo-premium')), pois mantém o estilo no CSS, facilita manutenção e garante consistência visual.
