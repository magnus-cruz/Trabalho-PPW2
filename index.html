Diferença entre .style e .classList:

Usar .style.backgroundColor aplica um estilo diretamente no atributo inline do elemento HTML, ou seja, funciona como escrever style="background-color: red" direto na tag — é pontual, sobrescreve tudo e não reutilizável. Já .classList trabalha com classes CSS já definidas na folha de estilos: ao adicionar ou remover uma classe com .toggle(), você ativa um conjunto inteiro de regras de uma vez, mantendo o estilo separado do código JavaScript — o que é mais organizado, fácil de manter e mais próximo das boas práticas do desenvolvimento web.
